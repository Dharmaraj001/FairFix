**FairFix **
FairFix is a full-stack MERN application designed to reduce disputes during minor vehicle accidents by providing a neutral damage cost range, nearby mechanic verification, and a digital on-road settlement record.

# Key Highlights
  *MERN stack(MongoDB,Express,React,Node.js)
  *Role-based system(Driver,Mechanic,Admin)
  *Geo-based mechanic discovery
  *Visual damage cost estimation(rule-based)
  *Digital settlement receipt generation
  *Built with real-world Indian traffic scenarios in mind
